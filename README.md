# 💸 Casharoo

A simple personal money management app built with Flutter.

---

## 📱 Features

* Add income and expenses
* View transactions
* Delete transactions
* Local storage using Hive

---

## 🛠 Tech

* Flutter
* Hive

---

## 🚀 Run

```bash
git clone https://github.com/your-username/Casharoo.git
cd casharoo
flutter pub get
flutter run
```
